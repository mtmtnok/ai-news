カテゴリ「半導体」の追加

  • カテゴリは現行6種＋半導体 の7種に。index.html の CAT_COLOR に1色追加（色は実装時に既存パレットと調整）。タブは作らない（同一フィード。過疎化と実装コストの理由で不採用）。
  • 守備範囲: AIチップ（NVIDIA/AMD/自社チップ）、ファウンドリ（TSMC/Samsung/Rapidus）、HBM・メモリ、製造装置（ASML等）、輸出規制・地政学、データセンター投資・電力。日本勢（Rapidus・キオクシア・東京エレクトロン等）はAI接点が薄くても拾う。車載・スマホSoC等のAI無関係トピックは対象外。

SNSバズの扱い

  • カテゴリは新設せず内容で分類（新モデル/ツール等）。要約の冒頭に「SNSで話題」と明記。
  • 未検証情報を含む場合は本文に「公式発表は未確認」と注記し、importance は 2 を上限とする。

§2. 企業相関図 — map.html（Phase 2）

別ページ map.html に vis-network（CDN読込み）でインタラクティブな網図を描く。データは relations_data.js に分離し、週次更新で育てる。

データモデル

window.RELATIONS = {
  nodes: [
    { id:"openai", label:"OpenAI", group:"ai_lab" },        // group: ai_lab / bigtech / chip / cloud / jp / vc
    { id:"nvidia", label:"NVIDIA", group:"chip" }, ...
  ],
  edges: [
    { from:"microsoft", to:"openai", type:"invest",         // type: invest / partner / talent / ma
      label:"累計約130億ドル出資", date:"2023-01",
      source:"https://..." }, ...                            // 全エッジにソースURL必須
  ]
};
エッジ種別線の表現トグル例
invest 出資・資金調達実線・太さ=金額規模画面上部のチェックボックスで種別ごとに表示ON/OFF。
「複雑なら図を分ける」要望は種別トグルで実現（実質4枚の図）。		Microsoft→OpenAI
partner 提携・供給契約破線NVIDIA→各社チップ供給、クラウド契約
talent 人材移籍・創業系譜点線大きな移籍・スピンアウトのみに限定（検証コストが高いため）
ma 買収・合併二重線相当（太実線）買収履歴
  • 操作: ノードドラッグ・ズーム・クリックで関係の詳細（日付・金額・ソースURL）をパネル表示。スマホはピンチズーム対応。
  • 導線: index.html ヘッダーに「🕸 相関図」リンク、map.html から戻るリンク。
  • 初期構築: 主要20〜30社（米フロンティアラボ・ビッグテック・半導体＋日本勢: SB/NTT/トヨタ/Rapidus等）、直近2年の主要ディール50〜80本をWeb検索で一括構築。全エッジに金額・時期・ソース付き。
  • 週次更新: 週1回、その週に追記したニュース（資金調達・提携・買収・大型移籍）から関係を抽出して relations_data.js に追記（Q1で曜日決定）。古いエッジは消さず履歴として残す。
